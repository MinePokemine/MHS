MH$
===
*By MinePokemine*

This Spigot/Paper plugin adds a currency called MH$ to Minecraft. It works like any other currency. You can use it with any mods that work with currency.

It's not a normal currency, though. Each 1,000 (or whatever you change it to in the configs) MH\$ you have is a heart you have. When one player kills another, the killer get MH\$1000 (Again configurable), and the victim loses the same amount.
